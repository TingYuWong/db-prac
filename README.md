# MongoDB + Express Bare-Bones

> Step 1: git clone

> Step 2: make sure have your mongodb run on localhost

> Step 3: set .env: DB_CONNECTION

> Step 4: INSERT data

> Step 5: yarn install & yarn dev

> Step 6: open http://localhost:3000/

## document structure

```
{
    index: 0,
    type: 'dog',
    breed: 'Havanese',
    name: 'Luna'
}
```
